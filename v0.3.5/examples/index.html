Physical Oceanography

  • ECCO_standard_plots.jl (➭ code link) : explore the Ocean climatology, currents, and more.
  • NSLCT_notebook.jl (➭ code link) : plot global mean sea level data from NASA and Dataverse.

Detail

ECCO_standard_plots.jl lets you explore climate indices and climatologies derived (via this and that) from gridded ocean climatologies. The data is retrieved from dataverse.org, and intermediate results from zenodo.org.

NSLCT_notebook.jl lets you access sea level data from NASA and Dataver portals (HTTP.jl, Dataverse.jl), organize it into tables (DataFrames.jl), and plot it (Makie.jl).

Note

For more on these estimates, and how to use them in Julia, please refer to the following documentation and links therein.

  • MeshArrays.jl : gridded Earth variables, domain decomposition, C-grid support; Ocean Circulation, Geography tutorials.
  • MITgcmTools.jl : framework to interact with MITgcm (setup, run, output, plot, etc) and ECCO output.
  • IndividualDisplacements.jl : simulation and analysis of materials moving through oceanic and atmospheric flows.

Marine Ecosystems

  • CBIOMES_climatology_plot (➭ code link) : visualize ocean colour and biomass climatologies
  • CBIOMES_climatology_create (➭ code link) : recreate the CBIOMES-global climatology files

The CBIOMES-global climatology (alpha version) is a global ocean state estimate that covers the period from 1992 to 2011 (ECCO). It is based on Forget et al 2015 for ocean physics MIT general circulation model and on Dutkiewicz et al 2015 for marine biogeochemistry and ecosystems Darwin Project model.

Note

For more notebooks involving CBIOMES and related efforts, take a look at the following pages.

  • JuliaCon2021 workshop : Modeling Marine Ecosystems At Multiple Scales Using Julia.
  • PlanktonIndividuals.jl : simulate the life cycle of ocean phytoplankton cells and their environment.
